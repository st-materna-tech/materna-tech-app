1. download the project directory
2. open the terminal and cd into the downloaded folder
3. install the packages using 'pip install -r requirements.txt' command
4. cd into the 'streamlit files' folder
5. run the application by typing 'streamlit run app.py' and wait for it to load the web page
6. it should automatically open the web page in the browser. 
	if it doesn't, copy the localhost address and paste it in the browswer to view the web page.
